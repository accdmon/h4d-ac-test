# php-laravel-hello-world
A sample code of PHP with Laravel

## Deploy this app to localhost

```
$ git clone https://github.com/accdmon/h4d-ac-test.git
$ cd h4d-ac-test/
$ composer install
$ php artisan serve
```

Access to

- http://127.0.0.1:8000/
- http://127.0.0.1:8000/hello/
- http://127.0.0.1:8000/world/
